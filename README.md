# QURRA

QURRA (Quranic Understanding: Reading Rules Adventure) adalah aplikasi pembelajaran tajwid
berbasis Nuxt 3 dan Phaser 3.

## Menjalankan proyek

```bash
npm install
npm run dev
```

## Pemeriksaan kualitas

```bash
npm run typecheck
npm run lint
npm run format:check
npm run build
```

Dokumen Spec-Driven Development tersedia di `.kiro/specs/qurra-game/`.
